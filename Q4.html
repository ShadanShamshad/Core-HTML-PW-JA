1. b: Tag It is used to make a text bold.

  2. i: Tag It is used to represent a text in some different voice.

  3. u: Tag It is used to render enclosed text with an underline.

  4. p: Tag It represents a paragraph in an HTML document.

  5. br Tag: br stands for break line, it breaks the line of the code.

  6. hr Tag: hr stands for Horizontal Rule. This tag is used to put a line across the webpage.

  7. aside Tag: It defines content aside from main content. Mainly represented as sidebar.

  8. button Tag: It is used to represent a clickable button

  9. canvas Tag: It is used to provide a graphics space within a web document.

  10. div Tag: It defines a division or section within HTML document.
